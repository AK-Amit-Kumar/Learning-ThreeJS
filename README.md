## 🚀 About This Repo

This repository is my **learning playground** for Three.js — the powerful JavaScript library that brings 3D graphics to the browser using WebGL.

Here you'll find experiments, mini-projects, and demos as I explore concepts like:

- 📦 Scenes, Cameras & Renderers
- 💡 Lighting (Ambient, Directional, Point & more)
- 🎨 Materials & Textures
- 🔷 Geometries (Box, Sphere, Torus, Custom meshes)
- 🎞️ Animations & the render loop
- 🖱️ Interactivity & Controls (OrbitControls, etc.)
- 🌍 3D environments and effects

> **Note:** This is a work in progress. Things will break, change, and (hopefully) get cooler over time. 🛠️

## 🛠️ Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [Git](https://git-scm.com/)

### 📥 Clone the Repository

```bash
git clone https://github.com/AK-Amit-Kumar/Learning-ThreeJS.git
cd Learning-ThreeJS
```

### 📦 Install Dependencies

```bash
npm install
```

### ▶️ Run the Dev Server

```bash
npm run dev
```

Then open your browser and visit:

```
http://localhost:5173
```

> The page will hot-reload automatically as you make changes. ✨
